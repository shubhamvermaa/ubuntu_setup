## Keyboard Shortcuts  
**Settings → Keyboard**

1. Increase key repeat speed and reduce delay  
2. Map **Ctrl + Tab** → **Alt + Tab**  
3. Move window to workspace (Shift + Left / Right)  
4. Switch workspaces left / right
